About eXo Cloud Workspaces

eXo Cloud Workspace is a private social intranet that makes it easy to connect and collaborate with your coworkers. Going beyond simple status updates, eXo Cloud Workspaces provides tools that let you capture, organize and act on your team’s internal knowledge, from a full-featured enterprise wiki, rich content management system, forums, chat and more.

What can you do with Cloud Workspace?

Features Overview

Web-based collaborative editor

eXo Cloud Workspace is a full web-based code editor allowing you to write code and run it directly in your browser.

With dedicated storage space for each account, your can access and work on your projects from any computer.

Invite up to 5 users to share you workspace and collaborate on your workspace.

Choose your language and framework: support for JavaScript, Java, Spring, Groovy, Ruby, PHP and more

With auto-completion and syntax highlighting for HTML, JavaScript, PHP, Java, Groovy, Ruby - and more coming soon - you can write apps in your language of choice.

Beyond interpreted languages, eXo Cloud IDE includes support for standard Java classes (servlets or POJOs) and Java Server Pages (JSP). It even allows you to build your Java applications in the Cloud using Maven.

Templates for OpenSocial gadgets, sample apps in Spring or Ruby on Rails will help you get started even faster.

Easy On-Ramp to Cloud Deployments: Built-in Support for Multiple Platform-as-a-Service (PaaS) Solutions

Run your apps in the cloud by deploying to your favorite PaaS directly from the IDE. The IDE allows developers to manage, edit and redeploy changes from the PaaS menu. Current support for:

CloudBees: for building, storing and running Java apps.

Cloud Foundry: for Spring, Rails or Node.js apps. Support for any Cloud Foundry-based target (CloudFoundry.com, Micro Cloud Foundry or third-party PaaS like Stackato and AppFog).

Heroku: for Java, Rails, and Node.js apps.

Red Hat OpenShift: for Java, PHP and Rails apps.

Native Git Integration for Version Control & Deployment Flexibility

eXo Cloud IDE features an easy-to-use Git menu, which natively integrates protocol commands. This gives developers access to Git controls, as well as visibility into the version history of repositories, without ever having to leave the eXo Cloud IDE domain.

The Git interface will let you clone, commit, or fork any Git repository. With the SSH key manager, you can configure SSH keys for different remote repositories. Review the commit history and compare with older versions.
